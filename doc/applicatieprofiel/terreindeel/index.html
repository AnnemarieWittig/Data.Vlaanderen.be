Objectencataloog terreindeel

Laatste aanpassing
2018-05-29

Dit document beschrijft een objectencataloog, in dit geval Objectencataloog terreindeel. Dit document geeft een overzicht van alle attributen die op een bepaald object gebruikt kunnen worden.

Licentie

Deze specificatie van Informatie Vlaanderen is gepubliceerd onder de "Modellicentie Gratis Hergebruik - v1.0".

Overzicht

In dit document wordt correct gebruik van de volgende entiteiten toegelicht:

Beschrijving
De wegbaan (Wbn) is de voorstelling van de wegcorridor in het GRB. De wegbaan wordt overlangs opgedeeld in verschillende arbitrair afgebakende wegbaanelementen. Er zijn twee types wegbaanelementen: een kruispuntzone komt overeen met een kruispunt; een wegsegment komt overeen met een wegvak.
