Samenvatting

Het Mandaat vocabularium specificeert een aantal klassen en eigenschappen om een mandaat te beschrijven.

Status van dit document

Deze sectie beschrijft de status van dit document op het moment van publicatie. Het kan zijn dat er ondertussen al nieuwere versies zijn uitgebracht.

Dit document is een specificatie van een vocabularium. Deze specificatie dient om de definitie, gebruik, domein en bereik van de RDF termen binnen in dit domein te verduidelijken.

Dit is een nieuwe versie uitgebracht door de Data Vlaanderen. Opmerkingen en vragen kunnen gestuurd worden naar de mailing lijst. De termen die hier zijn gedefinieerd kunnen worden vervangen door deze groep of vanuit andere groepen in de toekomst. De termen zullen echter nooit verdwijnen, noch zullen de definities veranderen behalve dan om een bestaande definitie verder te verduidelijken.

1. Licentie

Dit vocabularium gehost door Informatie Vlaanderen in samenwerking met Agentschap Binnenlands Bestuur (ABB) is uitgegeven onder de "Modellicentie Gratis Hergebruik - v1.0".

2. Conformiteit

Een uitwisseling van gegevens, op welke manier deze uitwisseling ook gebeurt, is conform aan dit vocabularium wanneer het de terminologie (klassen en eigenschappen) gebruikt op een manier die consistent is met de semantiek zoals opgesteld in de nieuwste versie van de specificatie (domein, bereik, definitie en gebruik) en het geen terminologie gebruikt uit andere vocabularia als alternatief voor de voorgestelde terminologie opgenomen in dit vocabularium.

3. Overzicht

Deze sectie somt alle klassen en eigenschappen van het vocabularium op.

4. Klassen

Deze sectie geeft een formele definitie aan elke klasse.

Klasse Bestuurseenheid

Type Klasse
URI http://data.vlaanderen.be/ns/mandaat#Bestuurseenheid
Definitie Publieke, bestuurlijke organisatie verbonden aan een territoriaal omschreven gebied met bepaalde verantwoordelijkheden waarbinnen het bestuurshandelingen kan stellen.

Klasse Bestuursorgaan

Type Klasse
URI http://data.vlaanderen.be/ns/mandaat#Bestuursorgaan
Definitie Een formeel orgaan van een rechtspersoon dat bevoegd is om rechtshandelingen te stellen.

Klasse Formeel Kader Aanstelling

Type Klasse
URI http://data.vlaanderen.be/ns/mandaat#FormeelKaderAanstelling
Definitie Het geheel van regelgeving, rechtshandelingen en beleidskeuzes waardoor de invulling van een of meerdere mandaten door een persoon vanaf een bepaalde datum (tijdelijk) start.

Klasse Formeel Kader Beëindiging

Type Klasse
URI http://data.vlaanderen.be/ns/mandaat#FormeelKaderBeeindiging
Definitie Het geheel van regelgeving, rechtshandelingen en beleidskeuzes waardoor de invulling van een of meerdere mandaten door een persoon vanaf een bepaalde datum (tijdelijk) beëindigd worden.

Klasse Fractie

Type Klasse
URI http://data.vlaanderen.be/ns/mandaat#Fractie
Definitie Een fractie is een deel van een rechtstreeks verkozen volksvertegenwoordigend orgaan waarvan de leden op inhoudelijk en logistiek vlak willen samenwerken. Het formeel kader bepaalt wie samen een fractie kan vormen, doorgaans bestaat ze uit volksvertegenwoordigers die tot dezelfde politieke partij of stroming behoren.

Klasse Kandidatenlijst

Type Klasse
URI http://data.vlaanderen.be/ns/mandaat#Kandidatenlijst
Definitie Lijst met kandidaten.

Klasse Mandaat

Type Klasse
URI http://data.vlaanderen.be/ns/mandaat#Mandaat
Definitie De bevoegdheid om gedurende een bepaalde periode een bestuursorgaan te vormen of er deel van uit te maken.

Klasse Mandataris

Type Klasse
URI http://data.vlaanderen.be/ns/mandaat#Mandataris
Definitie Een persoon in zijn rol als houder van een mandaat.

Klasse Rechtstreekse Verkiezing

Type Klasse
URI http://data.vlaanderen.be/ns/mandaat#RechtstreekseVerkiezing
Definitie De uitvoering van een formele procedure waarbij stemgerechtigde burgers bepalen aan welke persoon een mandaat kan worden toegekend.

Klasse Tijdsgebonden Entiteit

Type Klasse
URI http://data.vlaanderen.be/ns/mandaat#TijdsgebondenEntiteit
Definitie Stelt een entiteit voor gedurende een specifieke periode van zijn levensduur.

Klasse Verkiezingsresultaat

Type Klasse
URI http://data.vlaanderen.be/ns/mandaat#Verkiezingsresultaat
Definitie Individuele verkiezingsuitslag van een kandidaat.
