Samenvatting

Het Besluit vocabularium specificeert een aantal klassen en eigenschappen om een besluit te beschrijven.

Status van dit document

Deze sectie beschrijft de status van dit document op het moment van publicatie. Het kan zijn dat er ondertussen al nieuwere versies zijn uitgebracht.

Dit document is een officiële specificatie van een vocabularium. Deze specificatie dient om de definitie, gebruik, domein en bereik van de RDF termen binnen in dit domein te verduidelijken.

Dit is een nieuwe versie uitgebracht door de OSLO Werkgroepen. Opmerkingen en vragen kunnen gestuurd worden naar de mailing lijst. De termen die hier zijn gedefinieerd kunnen worden vervangen door deze groep of vanuit andere groepen in de toekomst. De termen zullen echter nooit verdwijnen, noch zullen de definities veranderen behalve dan om een bestaande definitie verder te verduidelijken.

1. Licentie en aansprakelijkheid

Dit vocabularium gehost door Informatie Vlaanderen is uitgegeven onder de "Modellicentie Gratis Hergebruik - v1.0".

2. Conformantie

Een uitwisseling van gegevens, hoe deze uitwisseling ook gebeurt, is conform aan dit vocabularium wanneer het de terminologie (klassen en eigenschappen) gebruikt op een manier die consistent is met de semantiek zoals opgesteld in de nieuwste versie van de specificatie (domein, bereik, definitie en gebruik) en het geen terminologie gebruikt uit andere vocabularia als alternatief voor de voorgestelde terminologie opgenomen in dit vocabularium.

3. Overzicht

Deze sectie somt alle klassen en eigenschappen van het vocabularium op.

Dit vocabularium verwijst verder naar termen uit de volgende vocabularia:

4. Klassen

Deze sectie geeft een formele definitie aan elke klasse.

Klasse Agendapunt

Type Klasse
URI http://data.vlaanderen.be/ns/besluit#Agendapunt
Definitie Formeel aanvaard te behandelen onderwerp op een zitting.

Klasse Artikel

Type Klasse
URI http://data.vlaanderen.be/ns/besluit#Artikel
Definitie Formeel afgebakend onderdeel van een besluit, dat een of meer van de beoogde rechtsgevolgen beschrijft.

Klasse Behandeling van Agendapunt

Type Klasse
URI http://data.vlaanderen.be/ns/besluit#BehandelingVanAgendapunt
Definitie De uitvoering van de acties om een agendapunt te verwerken tijdens een zitting.

Klasse Besluit

Type Klasse
URI http://data.vlaanderen.be/ns/besluit#Besluit
Definitie De authentieke schriftelijke neerslag van een beslissing van een bestuursorgaan. Deze beslissing houdt een rechtshandeling in waarbij sprake is van een beoogd rechtsgevolg.

Klasse Stemming

Type Klasse
URI http://data.vlaanderen.be/ns/besluit#Stemming
Definitie Geformaliseerde handeling waarbij de stemgerechtigde leden van een bestuursorgaan hun keuze te kennen geven.

Klasse Vergaderingsactiviteit

Type Klasse
URI http://data.vlaanderen.be/ns/besluit#Vergaderingsactiviteit
Definitie Een vergadering of onderdeel ervan.

Klasse Zitting

Type Klasse
URI http://data.vlaanderen.be/ns/besluit#Zitting
Definitie Een geformaliseerde samenkomst van de leden van een bestuurorgaan met het doel om de aangelegenheden te regelen waarvoor het bevoegd is.
